# V37: Entropy Compression Test

## Claim
Kolmogorov complexity distinguishes randomness types.

## Formula
```
K(x) ≈ |x|    → incompressible (bit-level)
K(x) << |x|   → compressible (physics-level)
```

## Result
**VERIFIED**
- True binary randomness: < 1% compressible
- Audio signals: 91.6% compressible

Cryptographic keys ≠ physical randomness.
