WHAT THESE RESULTS MEAN
================================================================================

  Ω(n log n) lower bound for comparison sorting
  Applies to: ALL n! permutations
  Size: n! = exponential

THE BOUNDED CASE:
  O(n) upper bound for d-bounded displacement
  Applies to: Only O(n^d) permutations (polynomial)
  Size: O(n^d) = polynomial for constant d

KEY INSIGHT:
  The classical bound doesn't break. The problem space shrinks.
  Bounded constraints reduce solution space from exponential to polynomial.

ALGEBRAIC EXPLANATION:

  Generated by adjacent transpositions
  Cayley distance metric from identity
  Information content: d log n bits (vs n log n for all perms)
